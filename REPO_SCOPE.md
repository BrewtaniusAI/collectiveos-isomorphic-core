# Repository Scope

This repository is the implementation-facing surface of the CollectiveOS isomorphic core.

## Includes
- Runnable code
- Conformance artifacts
- Implementation-level claims
- Reproducibility guidance

## Does not include
- The full theoretical corpus
- All architectural research artifacts
- Every deployment configuration

## External context
Broader theoretical and operational context is documented in associated publications, including:
- https://doi.org/10.5281/zenodo.19477170

## Intent
This repository is designed to:
- be executable
- be auditable
- be reproducible
- support scrutiny at the implementation level
